Сервис высотных данных

Сервис позволяет получить высотные данные по координатам одной точки или по координатам множества точек

  1. /elevation?data=[lat1,lng1] - получение высотных данных по одной точке в формате: {lat_origin: lat, lng_origin: lng, lat:lat, lng:lng, elevation: elevation}
  2. /elevation?data=[[lat1,lng1],[lat2,lng2], ..] - получение высотных данных по массиву точек в формате: [{lat_origin: lat, lng_origin: lng, lat:lat, lng:lng, elevation: elevation}, ...]
